# Cameo Faction Roster

Icon-grid view of every Cameo-mod faction — cameos grouped by production queue,
hover a tile for cost / HP / armour / speed / sight / power / weapon stats and
prerequisites. Live at **https://blackrobe.github.io/cameo-units/**.

Sibling of the [tech-tree graph view](../cameo-techtree/). Both are generated by
the same tool — see [`../cameo-techtree/README.md`](../cameo-techtree/README.md).
Regenerate from upstream with `..\cameo-techtree\update.cmd`, which rewrites
`data/units.json` and copies the on-disk PNG cameos into `icons/`.

## Icon coverage

~100% (2150+/2152). Cameos are rendered straight from the mod sprites (SHP and
PNG alike) by the engine utility command `--export-cameos` (see
[`../cameo-techtree/README.md`](../cameo-techtree/README.md#regenerating-cameo-icons)).
Units without an icon sequence fall back to a flat queue-coloured tile.

## Files

```
index.html / app.js / style.css   the grid view
data/units.json                   generated dataset (stats + icon refs)
icons/*.png                       copied cameo PNGs
```
